Ashesi University Academic Management System
--------------------------------------------

Description:
This Java application is designed to manage academic operations at Ashesi University, facilitating course management, student enrollment, and grade tracking.

Getting Started:
To run this program, you will need Java installed on your machine. Ensure you have the Java Development Kit (JDK) version 8 or higher.

Instructions:
1. Open your command line interface (CLI).
2. Navigate to the directory containing the program files.
3. Compile the Java files using the following command:
   javac *.java

Running the Program:
After compiling the files, you can run the program with the following command:
   java Main

This will start the program and display the main menu where you can choose to enroll in courses, drop courses, record grades, or generate a transcript.

Example:
> java Main

Follow the on-screen prompts to interact with the system. Enter numbers to select menu options and follow additional prompts for each operation.

Exiting the Program:
You can exit the program by selecting '5' from the main menu .

Note:
Ensure all Java files (Main.java, Student.java, Course.java, etc.) are in the same directory and compiled without errors before running the Main class.
